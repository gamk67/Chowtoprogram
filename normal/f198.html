Confusing Equality (==) andAssignment (=) Operators

      • Example, replacing == with =:

if ( payCode = 4 )

printf( "You get a bonus!\n" );

This sets payCode to 4

4 is nonzero, so expression is true, and bonus awarded no matter what the payCode was

    • Logic error, not a syntax error
    • Suggestion: write the condition the other way round if ( 4 == payCode ) .....This is correct, but if ( 4 = payCode ) .....is a syntax error, so it won't compile!
