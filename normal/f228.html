Some header files

<math.h> Contains function prototypes for

math library functions.

<setjmp.h> Contains function prototypes for

functions that allow bypassing of

the usual function call and return

sequence.

<signal.h> Contains function prototypes and macros

to handle various conditions that may

arise during program execution.

<stdarg.h> Defines macros for dealing with a list

of arguments to a function whose number

and types are unknown.

<stddef.h> Contains common definitions of types

used by C for performing certain calculations.
