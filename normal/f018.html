C Standard Library

  • C programs consist of pieces/modules called functions

    • A programmer can create her own functions

      • Advantage: the programmer knows exactly how it works
      • Disadvantage: time consuming

    • Programmers often use the C library functions

      • Use these as building blocks

    • Avoid re-inventing the wheel

      • If a pre-made function exists, it is generally best to use it rather than write your own
      • Library functions carefully written, efficient, and portable
