A Simple C Program:Text output

  • return 0;

    • A way to exit a function
    • return 0, in this case, means that the program terminated normally

  • Right brace }

    • Indicates end of main has been reached

  • Linker

    • When a function is called, linker software locates it in the library
    • Inserts it into object program
    • If function name is misspelled, the linker will produce an error because it will not be able to find it in the library
