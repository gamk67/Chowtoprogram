Increment and Decrement Operators

  • If c equals 5, then

printf( "%d", ++c );

    • Prints 6

printf( "%d", c++ );

    • Prints 5
    • In either case, c now has the value of 6

  • When variable not in an expression

    • Preincrementing and postincrementing have the same effect

++c;

printf( “%d”, c );

    • Has the same effect as

c++;

printf( “%d”, c );
