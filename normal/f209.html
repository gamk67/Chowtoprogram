ceil( x ) rounds x to the higher integer ceil( 9.2 ) is 10.0

ceil( -9.8 ) is -9.0

 

floor( x ) rounds x to the lower integer floor( 9.2 ) is 9.0

floor( -9.8 ) is -10.0

 

pow( x, y ) x raised to power y (xy) pow( 2, 7 ) is 128.0

pow( 9, .5 ) is 3.0

 

fmod( x, y ) remainder of x/y as a float fmod( 13.657, 2.333 ) is 1.992
