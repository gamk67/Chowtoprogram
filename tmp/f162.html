Error-Prevention Tip

Using the final value in the condition of a while or for statement and using the <= relational operator will help avoid off-by-one errors. For a loop to print the values 1 to 10, the condition should be counter <= 10 rather than counter < 11 or counter < 10.

Although the value of the control variable can be changed in the body of a for loop, this can lead to errors. It is best not to change it.
