Insertion sort

  • Insertion sort
    • Several passes through the array
    • Consider elements one at a time
    • Each new element is inserted into its correct position
    • Must make room by moving the other elements
    • After N passes N elements are in the right order
  • Pseudocode:

    for i=1 to n-1 do
        find the position of A[i] in A[0]..A[i-1]
        shift the array elements to make room
        insert A[i] into its correct position
