for Repetition Statement

  • For loops can usually be rewritten as while loops:
       initialization;
       while( loopContinuationTest ) {
           statement;
           increment;
       }
  • Initialization and increment
    • Can be comma-separated lists
    • Example:
      for (i = 0, j = 0; j + i <= 10; j++, i++)
          printf( "%d\n", j + i );
