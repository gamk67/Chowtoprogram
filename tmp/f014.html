Programming Languages

  • There are three types of programming languages:
    • Machine languages - Strings of numbers giving machine specific instructions. Example:
         1400593419
         1300042774
         1200274027
    • Assembly languages - English-like abbreviations representing elementary computer operations (translated via 'assembler' software). Example:
         LOAD  BASEPAY
         ADD   OVERPAY
         STORE GROSSPAY
