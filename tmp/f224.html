Function Prototypes

  • A function prototype contains
    • The function name
    • Parameters – what the function takes in
    • The return type – the data type that the function returns (default int)
    • The prototype is used to validate functions
    • It is only needed if the function definition comes after it is used in the program
    • The function with the prototype:
      int maximum( int x, int y, int z );
      • Takes in 3 ints
      • Returns an int
