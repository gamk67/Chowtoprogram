Searching Arrays

  • Binary search
  • For sorted arrays only
  • Compares the middle element with the key
    • If they are equal, a match has been found
    • If key < middle, it looks in the first half of the array
    • If key > middle, it looks in the last half
    • Repeat
  • Very fast; at most n steps are required, where
    2n > number of elements
    • A 30 element array takes at most 5 steps (25 > 30)
