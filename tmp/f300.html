Sorting Arrays

  • Sorting data
    • Sorting is an important computing application
    • Virtually every organization must sort some data
  • Bubble sort
    • Several passes through the array are made
    • Successive pairs of elements are compared
      • If they are in increasing order (or identical ), no change is made
      • If they are in decreasing order, the elements are exchanged
    • Repeat
  • Large elements “bubble” upwards
