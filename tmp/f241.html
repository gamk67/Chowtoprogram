Defining Arrays

  • When defining arrays, specify
    • The name
    • The type of the array elements
    • The number of elements
      arrayType arrayName[ numberOfElements ];
    • Examples:
      int c[ 10 ];
      float myArray[ 3284 ];
  • Defining multiple arrays of the same type:
    • The format is similar to regular variables
    • Example:
      int b[ 100 ], x[ 27 ];
