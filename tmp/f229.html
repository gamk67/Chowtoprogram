Some header files

 

<stdio.h>         Contains function prototypes for the 
                  standard input/output library functions,
                  and information used by them.
<stdlib.h>        Contains function prototypes for
                  conversions of numbers to text and
                  text to numbers, memory allocation,
                  random numbers, and other utility
                  functions.
<string.h>        Contains function prototypes for
                  string-processing functions.
<time.h>          Contains function prototypes and
                  types for manipulating the time and date.
