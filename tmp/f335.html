Random Number Generation

  • rand function
    • Load <stdlib.h>
    • Returns a "random" number between 0 and RAND_MAX (at least 32767)
    •   i = rand();
  • Pseudo random
    • Generates a preset sequence of "random" numbers
    • Generates the same sequence for every function call
  • Scaling
    • rand() % n returns a number between 0 and n - 1
    • Add 1 to make a random number between 1 and n
      1 + ( rand() % 6) is a random number between 1 and 6
