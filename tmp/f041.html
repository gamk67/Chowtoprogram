Another Simple Program:Addition

  • int integer1, integer2, sum;
  • Declaration (definition) of variables: locations in memory where a value can be stored
  • int (its type) means the variables can hold integers (-1, 3, 0, 47)
  • Variables have names (identifiers), such as integer1, integer2, sum.
  • They consist of letters, digits (cannot begin with a digit) and underscores( _ ). They are case sensitive
  • Definitions appear before executable statements, otherwise it produces a syntax (compiler) error
