Computer Organization

  • There are six logical units in every computer:
    • Input unit - Obtains information from input devices
    • Output unit - Outputs information (e.g. screen, net)
    • Memory unit - Rapid access, low capacity, stores data
    • Arithmetic and logic unit (ALU) - Performs arithmetic calculations and logic decisions
    • Central processing unit (CPU) - Supervises and coordinates the other sections of the computer
    • Secondary storage unit - e.g. hard drive
