Passing Arrays to Functions

  • To pass an array argument to a function, specify the name of the array without any brackets
       int myArray[ 24 ];
       myFunction( myArray, 24 );
  • The array size is usually passed to the function
  • The name of the the array is the address of the first element
  • The function knows where the array is stored, and modifies the original memory locations
  • To pass an array element, pass the subscripted name (e.g. myArray[ 3 ]) to the function
