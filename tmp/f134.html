Increment and Decrement

  • Increment operator (++)
    • Can be used instead of c += 1
  • Decrement operator (--)
    • Can be used instead of c -= 1
  • Preincrement
    • The operator is used before the variable (++c or --c)
    • The variable is changed before the expression is evaluated
  • Postincrement
    • The operator is used after the variable (c++ or c--)
    • The expression executes before the variable is changed
