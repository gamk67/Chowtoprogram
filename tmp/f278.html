Character arrays

  • String “first” is really a array of characters
  • Character arrays can be initialized using string literals
       char string1[] = "first";
  • The null character '\0' terminates a string
  • string1 actually has 6 elements. It is equivalent to
       char string1[] = { 'f', 'i', 'r', 's', 't', '\0' };
  • You can access individual characters:
       string1[ 3 ] is character ‘s’
  • The array name is the memory address of the array, so & is not needed for scanf:
       scanf( "%s", string2 );
    • this reads characters until a whitespace is encountered
    • Be careful not to write past end of the array, as it is possible to do so
