A Simple Program:Text output

  • return 0;
    • A way to exit a function
    • return 0, in this case, means that the program terminated normally
  • Right brace }
    • Indicates that the end of main has been reached
  • Linker
    • When a function is called, linker software locates it in the library, and inserts it into the object program
    • If the function name is misspelled, the linker gives an error message
