Another Simple Program:Addition

  • scanf( "%d", &integer1 );
  • Obtains a value from the user. It uses standard input (the keyboard)
  • This statement has two arguments:
    • %d indicates that data should be a decimal integer (conversion specifier)
    • &integer1 is the location in memory to store the data (address operator)
  • The user responds to the scanf statement by typing in a number, and then pressing the enter (return) key
