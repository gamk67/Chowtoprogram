Array Examples

  • Initialisers
    int n[ 5 ] = { 1, 2, 3, 4, 5 };
    • If there are not enough initialisers, the rightmost elements become 0
    • int n[ 5 ] = { 0 };
      • All elements are set to 0
    • If there are too many initialisers, a syntax error occurs
    • C arrays have no bounds checking
  • If the size is omitted, the initialisers determine it
    int n[ ] = { 1, 2, 3, 4, 5 };
    • 5 initialisers, therefore this is a 5 element array
