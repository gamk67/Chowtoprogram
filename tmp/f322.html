Multiple-Subscripted Arrays

  • Multiple subscripted arrays
    • Are like tables with rows and columns (m by n array)
    • Treat them like matrices: specify a row, then a column
  • Initialisation
    • int b[ 2 ][ 2 ] = { { 1, 2 }, { 3, 4 } };
    • The initialisers are grouped by row in brackets
    • If there are not enough, the unspecified elements are set to zero
          int b[ 2 ][ 2 ] = { { 1 }, { 3, 4 } };
  • Referencing elements
    • Specify the row, then column: printf( "%d", b[ 0 ][ 1 ] );
