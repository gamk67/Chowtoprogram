   This becomes:


  printf(“Entering while loop\n”);
  while ( grade != -1 ) {
     printf(“start while, grade=%d\n”, grade);
     total = total + grade;
     counter = counter + 1;
     printf(“total=%d, count=%d\n”, total, count);
     /* get next grade from user */
     printf( "Enter grade, -1 to end: " );
     scanf("%d", &grade);
  } /* end while */
  printf(“Exit while loop\n”);
The added printf statements print the values of the important variables as the loop progresses. We also print when we enter and exit the loop. If we never exit the loop, we must be going round inside the loop endlessly, so there has to be a problem with the loop control (which is a very typical problem).
